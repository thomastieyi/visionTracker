# **App Name**: VisionTrack

## Core Features:

- User Account Management: Create and manage user accounts to store individual vision data.
- Distance Recording: Record left and right eye test distances in centimeters.
- Vision Calculation: Automatically calculate vision 'degrees' (100/distance) for each eye.
- Historical Data View: View a chronological history of all recorded test results.

## Style Guidelines:

- Primary color: Soothing green (#A7D1AB) for a calm and trustworthy feel.
- Background color: Light gray (#F0F0F0) to provide a clean and unobtrusive backdrop.
- Accent color: A slightly darker green (#84A98C), analogous to the primary, for interactive elements and highlights.
- Body and headline font: 'PT Sans' for a modern and friendly user experience.
- Use clear, minimalist icons to represent different data points and functions.
- Maintain a clean, intuitive layout with ample spacing for easy readability.
- Incorporate subtle transitions and animations to provide feedback and enhance usability.
